# Credit Scoring Model

## Overview

This project predicts whether a customer is a **Good Credit Risk** or **Bad Credit Risk** using Machine Learning. The model was trained on the German Credit Dataset and deployed as an interactive web application using Streamlit.

## Problem Statement

Financial institutions need reliable methods to assess customer creditworthiness before approving loans. This project uses historical financial information to classify customers as good or bad credit risks.

## Dataset

**German Credit Dataset (Statlog)**

* 1000 customer records
* 24 financial attributes
* Binary classification problem
* Target Variable:

  * 1 = Good Credit Risk
  * 0 = Bad Credit Risk

## Machine Learning Model

**Random Forest Classifier**

The model was trained using the German Credit Dataset and evaluated using standard classification metrics.

## Model Performance

| Metric    | Score  |
| --------- | ------ |
| Accuracy  | 81.50% |
| Precision | 82.10% |
| Recall    | 94.33% |
| F1 Score  | 87.79% |
| ROC-AUC   | 82.05% |
